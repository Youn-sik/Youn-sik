# gstack 자연어 매핑 가이드

> Claude Code의 CLAUDE.md에 추가하여 슬래시 커맨드 없이 자연어로 gstack 스킬을 호출할 수 있도록 하는 매핑 규칙.

---

## gstack 자연어 호출 규칙

사용자가 아래 자연어 패턴으로 요청하면, 해당 gstack 스킬을 자동으로 실행한다.
`/browse` 스킬을 모든 웹 브라우징에 사용하고, `mcp__claude-in-chrome__*` 도구는 절대 사용하지 않는다.

```markdown
## gstack

gstack 스킬을 자연어로도 호출한다. 사용자가 코드 리뷰, QA, 배포, 보안 점검, 
아키텍처 검토, 디버깅, 회고 등을 자연어로 요청하면 해당 gstack 스킬을 자동 실행한다.

자연어 → 스킬 매핑:
- 코드 리뷰/버그 찾기/변경사항 감사 → /review
- 보안 점검/취약점/위협 모델링 → /cso  
- QA/테스트/유저 플로우 검증 → /qa
- 테스트만 하고 수정 말기 → /qa-only
- 배포/PR 올리기/머지 준비 → /ship
- PR 머지 + 프로덕션 배포 → /land-and-deploy
- 배포 후 모니터링 → /canary
- 성능 측정/Core Web Vitals → /benchmark
- 아키텍처/설계 검토/다이어그램 → /plan-eng-review
- 비즈니스/제품 방향성 검토 → /plan-ceo-review
- 기획/아이디어 논의/디자인 문서 → /office-hours
- 디자인 평가/AI 슬롭 → /plan-design-review
- 디자인 시스템/목업 → /design-consultation
- 디자인 감사 + 코드 수정 → /design-review
- 전체 플랜 리뷰 자동화 → /autoplan
- 버그 원인 추적/디버깅 → /investigate
- 문서 업데이트/README 갱신 → /document-release
- 주간 회고/스프린트 정리 → /retro
- 세컨드 오피니언/Codex 리뷰 → /codex
- 브라우저 열기/스크린샷 → /browse
- 쿠키 임포트/인증 세션 → /setup-browser-cookies
- 조심해서/안전 모드 → /careful
- 편집 범위 제한 → /freeze
- 프로덕션 최대 안전 → /guard
- 잠금 해제 → /unfreeze
- 배포 환경 설정 → /setup-deploy
- gstack 업데이트 → /gstack-upgrade

웹 브라우징은 항상 /browse 스킬을 사용한다. mcp__claude-in-chrome__* 도구는 사용하지 않는다.
```

## 참고

- 이 매핑은 gstack v25.3 기준 28개 전체 스킬을 포함합니다.
- 자연어 트리거는 한국어 + 영어 혼용을 고려하여 작성되었습니다.
- `disable-model-invocation` 설정이 `true`인 스킬은 자동 호출이 차단될 수 있으므로, 해당 설정을 `false`로 변경하거나 CLAUDE.md 매핑을 통해 우회합니다.
- 스킬 description 문자 예산은 컨텍스트 윈도우의 2%이므로, `/context` 명령으로 제외된 스킬이 없는지 주기적으로 확인하세요.
